Customizing coordinate axes

Check which coordinate axes you want to use for this example chart:

This example shows how to add different coordinate axes to the flot chart.

After you referenced the jquery.flot.coordinate.js, you can pass different coordinate name to the options object of the flot, and then the plugin will draw the axes for you.

You can use different coordinate axes at the same time!
